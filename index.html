Say Cheese!

Minimal javascript library for integrating a webcam and snapshots into your app.

Usage

After including the script, making use of *Say Cheese!* is simple :)

{% highlight javascript %} $(function() { var sayCheese = new SayCheese('#container-element'); sayCheese.on('start', function(evt, snapshot) { this.takeSnapshot(); }); sayCheese.start(); }); {% endhighlight %}
